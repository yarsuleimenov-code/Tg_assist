# Ключевые навыки

## Business Analysis

- выявление и структурирование требований;
- описание as-is / to-be процессов;
- user stories, use cases, acceptance criteria;
- backlog и постановка задач разработке;
- MVP scope и критерии готовности;
- описание ролей пользователей, сценариев, ограничений и рисков;
- подготовка developer handoff и business handoff.

## Product Thinking

- связь требований с бизнес-ценностью;
- выбор простого решения без лишней архитектуры;
- приоритизация MVP по основному пользовательскому сценарию;
- проектирование broker-first, client-facing и mobile-first workflows;
- фиксация ограничений MVP и roadmap production version;
- разделение test-mode prototype, visual specification и production scope.

## System Analysis

- проектирование сущностей, статусов и lifecycle;
- data contracts и field-level rules;
- status mapping и state logic;
- immutable snapshots для estimate/trip/report сценариев;
- auditability и traceability расчетов;
- описание API expectations и integration boundaries;
- работа с edge cases и validation rules.

## Data Analytics & Reporting

- SQL, Excel, Power BI, KPI analysis, dashboards, operational reporting;
- ABC/XYZ сегментация клиентской базы;
- анализ продаж, маржинальности, конверсии и среднего чека;
- проектирование отчетных представлений и расчетных таблиц;
- A/B-test analysis и интерпретация результатов;
- data quality audit и проверка расчетов на простых кейсах.

## Operations & Logistics Domain

- pricing logic для logistics / delivery workflows;
- broker calculator, quote, estimate, invoice, order lifecycle;
- warehouse loading/unloading workflows;
- piece-level tracking;
- eBOL / POD concepts;
- route, vehicle, packaging, fragile/non-stackable items и operational constraints;
- customer-facing order tracking.

## Technical Literacy

- Python basics для MVP и Telegram bot;
- aiogram, DeepSeek/OpenAI-compatible API, dotenv;
- HTML, CSS, JavaScript;
- React, Vite, TypeScript basics;
- Google Apps Script и Google Sheets как быстрый backend для MVP;
- Markdown documentation;
- GitHub, GitHub Pages, Docker, Google Cloud VM;
- localStorage, JSON backup/export-import, simple smoke tests.

## Documentation & Communication

- README для HR, бизнеса и технического интервьюера;
- architecture docs;
- data contracts;
- test cases и UAT cases;
- setup guides;
- current-state analysis;
- implementation plans;
- risk/limitation sections;
- краткая деловая коммуникация без лишнего усложнения.
