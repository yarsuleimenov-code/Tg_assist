# Кейсы

Статус: demo knowledge base. Перед публикацией замените кейсы на реальные материалы кандидата.

## Кейс 1. Scope MVP для AI-ассистента портфолио

Бизнес-вопрос:
Как быстро показать HR и техническому интервьюеру релевантный опыт кандидата без длинного ручного объяснения?

Пользователь:
HR, рекрутер или интервьюер.

Основной сценарий:
Пользователь задает вопрос в Telegram. Бот находит релевантные материалы в локальной knowledge base и формирует краткий ответ.

Критерии готовности:
- бот отвечает на текстовые вопросы;
- есть команды для проектов, навыков и ссылок;
- ответы основаны только на подготовленных Markdown-файлах;
- проект можно запустить локально и в Docker.

Риски:
- простая keyword-релевантность может пропустить сложный вопрос;
- demo knowledge base требует замены на реальные факты;
- история сообщений хранится только в памяти процесса.

## Кейс 2. Анализ CRM-воронки

Бизнес-вопрос:
Почему часть лидов теряется между первым контактом и квалификацией?

Подход:
- описать текущий процесс;
- определить роли и точки передачи;
- найти статусы без владельца;
- предложить минимальные изменения в процессе.

Результат:
- список проблем;
- целевой процесс;
- backlog улучшений;
- критерии приемки для изменений CRM.
