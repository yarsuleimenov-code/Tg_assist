# Ключевые навыки

Статус: demo knowledge base. Перед публикацией замените список на реальные навыки кандидата.

## Business Analysis

- выявление и структурирование требований;
- интервью с заинтересованными сторонами;
- описание бизнес-процессов;
- user stories и acceptance criteria;
- scope MVP;
- анализ рисков, ограничений и зависимостей.

## Product Thinking

- связь требований с бизнес-ценностью;
- приоритизация задач;
- формулирование пользовательских сценариев;
- определение критериев готовности;
- работа с гипотезами и метриками.

## Technical Literacy

- базовое понимание API и интеграций;
- работа с Markdown-документацией;
- GitHub-ready оформление проекта;
- Docker basics;
- Python basics для прототипирования.

## Communication

- деловая письменная коммуникация;
- структурные отчеты;
- краткое объяснение сложных решений;
- подготовка материалов для HR, бизнеса и технического интервью.
